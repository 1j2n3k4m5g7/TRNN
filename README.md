Quick reminder how to run after download:
1. download python 3.11.9
2. python files should be in folder named `py_3.11.9.x64` in same place as rest of this project (you need this to run `setup.bat`)
3. run setup.bat
4. you should be ready to double-click runner.bat and work

to edit see `script.py`
